# 🍞 BeriBakes Bakery Management System

A web-based bakery management system designed to help bakeries manage products, orders, customers, inventory, and financial records efficiently.

The system provides both **customer-facing e-commerce functionality** and a **powerful administrative dashboard** for managing bakery operations.

---

## 📌 Project Overview

The **BeriBakes Bakery Management System** automates bakery operations that are traditionally handled manually.  
It allows customers to browse bakery products, add items to a cart, and place orders online while administrators manage the business through a centralized dashboard.

This system improves:

- Order processing efficiency
- Product and inventory management
- Customer record management
- Financial transaction tracking
- Sales reporting and monitoring

## 🚀 How to Run the Project

### 1️⃣ Install XAMPP
Download and install XAMPP if not already installed.

### 2️⃣ Copy Project Folder
Place the project folder inside:


C:\xampp\htdocs\


### 3️⃣ Start Apache and MySQL

Open **XAMPP Control Panel** and start:

- Apache
- MySQL

### 4️⃣ Import the Database

Open browser and go to:


http://localhost/phpmyadmin


Then:

1. Click **Import**
2. Select the file:


database/beribakes.sql


3. Click **Go**

---

### 5️⃣ Run the System

Open your browser and go to:


http://localhost/beribakes
